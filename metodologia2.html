Statue of Ulisse Dini

Metodology

Tools Used

To re-create the Statue of Ulisse Dini, I used some tools:

  • XnView
  • Visual SFM
  • Meshlab
  • Blender


Acquisition

The 3D model has been realized taking a lot of photos on field; in order to catch all the details or features of the statue. The camera used to takes the photos is a Canon EOS 600D equipped with a Canon EFS 18-55mm. In order to have enought photo to recreate the 3D model I had to make 2 shoot. I've been taking photo for 2 hours for each shoot. Furthermore, has been used for the shoot a ladder. After the Acquisition the photos has been downloded into the PC.



To take the photos I used a Canon, a ladder and a lot of patient. After that I downloded the photo into the pc. And I lanched Visual SFM.



XnView

After the Acquisition, the photo has been checked. I noticed that the quality of the photos are too high. If I lanched the program using pictures with so hight resolution VisualSFM probably would crash in a second. In order to avoid this bug I change the resolution of the photos from 5184x3456 to 3000x2000. Before has been lanched Visual SFM, the photos that could create problems to the software has been descarted. For example the photo that has been descrated are the blurry photos or the photos with no focus.








Visual SFM

The photos then are loaded in Visual SFM. In total 200 photos were loaded. After that, apply the Compute Missing Matched that extrapolate the photos's feature and create a match between every single photos.


Compute 3D Reconstruction

Visual SFM has found the match beetween the photos, now using this procedure can create a 3D reconstruction of the scene. This procedure extrapolate informations from the match that has found in the Compute Missing Matched and estrapolate some information like: calibration, orientation and position in space of the camera.


Dense Model

After the "Compute 3D Reconstruction", the final step is: Dense Reconstruction. It's take 4 hours to have a dense clouds point reconstruction. This procedure create a dense point cloud. Visual SFM has created 5 cloud for the statue. The dense cloud point obtained from Visual SFM are too "dirty" so it's necessary to clean the dense reconstruction. Prior to that, the 5 dense reconstruction has been loaded at the same time into the program in order to have a complete dense model. For merge the 5 dense reconstruction in meshlab I used the filter called "Flatte Visible Layer"

Poisson Reconstruction

After that the statue has been cleaned, in order to create a uniform surface I apply the filter that's is called "Reconstruction: Poisson" using for the parameter "Octree Depth" and "Solver Divider" the same value: 12 in order to mantein the highter level of detail.




Manupulator Tools and Scale


Then I edited the orientation of the statue using the "Manipulator Tool", using this tool I can fixed the orientetion of the model in the axis x,y,z. Using the Transform: Scale, I scaled the 3D Model, so the model have a real unit of measurement.










Smoothing with Meshlab and Blender


After that the model that I've got is a model with a lot of impurities so I used a tools that it's is in meshlab that it's called "smooth". Using this tools I "clean" all the impurities of the statue. Another tools used is the sculptur mode in Blender: using the sculpture brush I can smooth all the impurities.













Color

After that in order to transert the color form a original dense cloud point to the poisson model, has been used another tools in meshlab that is called "Vertex Attribute Transfer."








This is the resultant model afetr apply the vertex attribute transfer.

Some problems...





I met some problem during all the process.The first problem that I met was during the Poisson transform, becouse I have in the mesh some hole that I can't fill so I decided to try some way to cover up the hole.












The first attempt was to mesh the hole of the base with a parallelepiped create with blender.

This attempt fails, so I decide to take a different approach: I went back to work in the clouds point and I clean all. After that I apply the Poisson Transformation.

To smooth all this "swelling" I used the sculpt mode in blender.
